# 貓咪影像分割及實際測量學任務

## 專案介紹
本專案從 COCO 2017 資料集中篩選出含有兩隻貓咪的圖片，共取得 16 張作為測試資料，並使用 CVAT 手動標註每隻貓的左眼與右眼作為驗證基準。此專案的任務如下：
1. 分割貓咪輪廓
2. 分割貓咪眼睛
3. 計算每隻貓咪的眼睛距離
4. 計算兩隻貓咪右眼的距離

## 評估模型

### 1. 貓咪輪廓分割：Intersection over Union, IoU

在分割貓咪輪廓任務中，模型利用預測的貓咪 Mask 與 COCO 資料集的 Ground Truth (GT) Mask 來計算重疊程度進行評估。其數學公式定義如下：

$$IoU = \frac{|A \cap B|}{|A \cup B|} = \frac{TP}{TP + FP + FN}$$

**公式參數解析：**
*   $A$：模型預測的貓咪輪廓像素集合 (`pred_mask`)。
*   $B$：真實標籤的貓咪輪廓像素集合 (`gt_mask`)。
*   $|A \cap B|$ (**Intersection**)：預測正確的貓咪像素數量（真正例，$TP$）。
*   $|A \cup B|$ (**Union**)：預測與真實標籤所涵蓋的總像素聯集數量（真正例 $TP$ + 偽正例 $FP$ + 偽負例 $FN$）。

---

### 2. 貓咪眼睛分割：關鍵點命中率 (Keypoint Hit Rate)

貓咪眼睛分割任務採用了一種基於**空間容忍度邊界框（Bounding Box Tolerance）**的命中率判定法。當模型生成的眼睛 Mask 包含真實眼眶中心即判定為該關鍵點「命中（Hit）」。

單張圖片中，特定眼睛關鍵點的命中函數 $H(g, M)$ 定義如下：

$$H(g, M) = \begin{cases} 1, & \text{if } \exists (x, y) \in M \text{ such that } \max(|x - g_x|, |y - g_y|) \le \tau \\ 0, & \text{otherwise} \end{cases}$$

整體驗證資料集的**眼睛關鍵點總命中率 (Total Hit Rate)** 公式為：

$$\text{Hit Rate} = \frac{\sum_{i=1}^{N} \sum_{k \in K} H(g_{i,k}, M_i)}{\text{Total Valid Keypoints}}$$

**公式參數解析：**
*   $g = (g_x, g_y)$：真實的眼睛關鍵點座標（包含 `c1_left`, `c1_right`, `c2_left`, `c2_right`）。
*   $M$：SAM 最終經由 45% Face ROI 與面積過濾後篩選出的眼睛預測 Mask 集合 (`final_masks`)。
*   $\tau$：空間容忍度閾值，本專案實作設定為 $\tau = 25\text{ px}$（即以關鍵點為中心建立 $50 \times 50\text{ px}$ 的搜尋鄰域）。
*   $N$：評估的總圖片張數。
*   $K$：有效眼睛關鍵點集合（自動排除未標記的 `NaN` 值）。

---

### 3. 距離測量：平均絕對誤差 (Mean Absolute Error, MAE)

在兩個計算距離的任務裡，透過預測座標與 XML 標註的真實值進行比對。公式如下：

#### A. 單貓個體內眼距誤差 (MAE-Intra)
用於評估單隻貓咪「左眼至右眼」距離的預測精準度（對應程式碼中的 `E1` 與 `E2`）：

$$MAE_{Intra} = \frac{1}{N_{cat}} \sum_{i=1}^{N_{cat}} |d_{pred, i} - d_{gt, i}|$$

*   $d_{pred, i}$：第 $i$ 隻貓咪模型預測出的雙眼像素距離（$\text{dist}$）。
*   $d_{gt, i}$：第 $i$ 隻貓咪真實標註的雙眼像素距離。
*   $N_{cat}$：所有影像中成功配對且擁有真實標註的貓咪總隻數。

#### B. 多貓個體間右眼距誤差 (MAE-Inter)
用於評估多貓場景下，「兩隻貓咪右眼之間」跨個體幾何間距的預測精準度（對應程式碼中的 `E_Inter`）：

$$MAE_{Inter} = \frac{1}{N_{pair}} \sum_{j=1}^{N_{pair}} |D_{pred, j} - D_{gt, j}|$$

*   $D_{pred, j}$：第 $j$ 組雙貓場景中，模型預測的「貓 1 右眼」與「貓 2 右眼」之歐幾里得距離：
    $$D_{pred} = \|\mathbf{p}_{rcat1} - \mathbf{p}_{rcat2}\|_2$$
*   $D_{gt, j}$：第 $j$ 組雙貓場景中，真實標註的兩隻貓右眼間距。
*   $N_{pair}$：資料集中含有雙貓以上且標註了間距的有效影像總組數。
